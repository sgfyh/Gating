# 基于 PVDF-压阻双通道床垫传感的睡眠呼吸信号质量门控方法

## 摘要草稿

床垫式非接触睡眠监测能够在无穿戴约束条件下采集呼吸相关微动信号，但实际夜间数据容易受到翻身、姿态改变和短时体动的影响，导致呼吸率估计不稳定。本文提出一种面向 PVDF-压阻双通道床下传感器的信号质量门控方法。该方法利用 PVDF 通道提取动态呼吸成分，并利用平行布置的压阻通道辅助表征接触状态变化和体动扰动；通过通道内电压变化率、包络变化率、小波细节能量和鲁棒统计阈值构建体动分数，再结合 PVDF 窗口级呼吸频带能量与峰间期稳定性形成信号质量指数。初步夜间实验表明，该方法能够自动剔除体动和低质量窗口，为后续呼吸率估计提供更稳定的数据基础。

关键词：床垫传感；PVDF；压阻传感；呼吸监测；信号质量评价；体动伪迹

## 论文主线判断

这个方向合理，而且比单纯“做一个呼吸率算法”更适合你当前数据状态。原因是：

- 床下传感的最大问题不是安静片段算不出呼吸，而是整晚数据里有大量姿态变化和体动伪迹。
- PVDF 与压阻通道天然互补：PVDF 适合呼吸/BCG 动态信号，压阻不作为呼吸通道，而作为接触状态、姿态改变和大体动变化的辅助门控通道。
- 质量门控不需要一开始就有临床级疾病标签，可以先用体动标注、窗口通过率、呼吸率稳定性做小论文。
- 后续可以自然扩展到多姿态鲁棒性、睡眠分期前处理、呼吸暂停事件候选片段筛选。

一句话创新点可以写成：

> 提出一种面向 PVDF-压阻双通道床下传感器的无监督信号质量门控方法，通过动态压电信号与准静态压阻信号的互补融合，实现夜间呼吸监测中的体动伪迹剔除和高质量窗口筛选。

## 方法

### 1. 数据采集与预处理

系统由床下 PVDF 通道和压阻通道组成，原始 ADC 采样率为 500 Hz。首先剔除超出 ADC 范围或非有限值的异常点，并采用线性插值补齐；随后将 ADC 值转换为电压。为降低计算量并匹配呼吸信号频段，采用 FIR 零相位降采样将信号从 500 Hz 降至 50 Hz。

### 2. PVDF 呼吸成分提取

对 PVDF 降采样信号进行 DB5 五层小波分解，并使用最高频细节分量估计噪声水平，对细节分量进行软阈值去噪。保留近似分量作为呼吸候选信号，再经过 0.10-0.60 Hz 呼吸频带滤波，得到 PVDF 呼吸波形。

### 3. 双通道体动分数

对每个通道分别计算三类变化特征：

- 电压变化率：对 0.3 s 平滑后的信号求绝对一阶差分；
- 包络变化率：以 8 s 趋势作为基线，计算交流分量绝对值的 1 s 平滑包络，再求包络变化率。
- 小波细节能量：对降采样信号进行 DB5 小波分解，重构非呼吸高频细节并计算 1 s 能量，用来描述短时宽带扰动。

每个特征采用中位数和 MAD 进行鲁棒标准化，并截断为非负分数。单通道体动分数为三类特征的加权和：

```text
S_c(t) = 0.4 Z(|d x_c / dt|) + 0.4 Z(|d env_c / dt|) + 0.2 Z(E_wavelet)
```

双通道融合分数为：

```text
S(t) = 0.5 S_PVDF(t) + 0.5 S_PR(t)
```

融合分数经过 1 s 平滑后，采用自适应阈值：

```text
T = max(3, median(S) + 3 * 1.4826 * MAD(S))
```

超过阈值的采样点被判定为体动候选，并向前后各扩展 2 s，形成最终体动门控掩码。

### 4. 窗口级 SQI

采用 30 s 滑动窗口、5 s 步长。每个窗口计算：

- 异常 ADC 比例；
- 体动掩码占比；
- PVDF 呼吸频带能量占比；
- 呼吸频谱主峰优势度；
- 峰间期估计呼吸率；
- 峰间期变异系数；

根据这些特征得到窗口质量分数 SQI，并分为 `good`、`usable`、`motion`、`low_quality`、`invalid`。后续呼吸率统计只使用通过门控的 `good/usable` 窗口。

## 初步实验结果

使用 `F:\双通道睡眠实验\2026_0413晚\数据` 中 20 个连续夜间 CSV 文件进行批量测试，总时长约 9.77 h。

| 指标 | 结果 |
| --- | ---: |
| 总时长 | 9.77 h |
| 加权窗口通过率 | 79.03% |
| 加权体动占比 | 13.25% |
| 单文件通过窗口中位呼吸率的中位数 | 14.14 bpm |
| 最稳定片段 `20260414_033313.csv` 通过率 | 97.75% |
| 最稳定片段体动占比 | 2.58% |
| 最稳定片段通过窗口中位呼吸率 | 13.73 bpm |

这些结果说明该门控方法能够识别夜间数据中的明显体动片段，并保留较大比例的稳定呼吸窗口。尤其在 03:33 附近的稳定睡眠片段中，门控几乎保留全部高质量窗口，估计呼吸率与已有整晚分段统计一致。

## 消融与对比实验设计

正式成文时建议补三组实验。

1. 门控前后对比  
   比较不加门控与加门控后呼吸率均值、标准差、CV 和异常窗口比例。

2. 通道与特征消融  
   比较 PVDF-only、PR-only、PVDF+PR，以及是否加入小波细节能量。预期结果是 PVDF 对短时冲击敏感，PR 对姿态和接触变化敏感，融合后更稳；小波细节能量应主要改善短时宽带扰动的识别。

3. 人工标注验证  
   选取 30-60 min 数据，人工标注体动时间段，计算门控掩码的 precision、recall、F1 或 IoU。

## 目前还缺什么

现在已经有方法和初步数据，但还不建议直接投稿。最需要补的是“评价依据”：

- 至少一段人工体动标注，证明门控不是只是在画漂亮图。
- 一个参考呼吸率来源，例如人工数呼吸、胸腹带、视频标注或你已有的稳定片段基准。
- 多姿态或多晚数据，证明不是只对单个片段有效。

补完这些后，小论文可以收敛成“系统实现 + 双通道 SQI 门控 + 初步夜间验证”的结构。
