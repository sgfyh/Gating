# 专利创新点梳理：基于压电-压阻复合条带的非接触式呼吸事件监测

## 1. 目标定位

本项目目标不是单纯输出呼吸率，而是面向睡眠场景下的呼吸事件监测，包括呼吸暂停、疑似低通气、体动干扰剔除以及呼吸信号可信度评估。

现有单条传感器结构为一条约 800 mm 的长条式集成模组，内部包含一路 PVDF 压电信号和一路压阻信号，二者平行布置。已有数据表明，该结构能够持续获得呼吸相关波形，并可利用压阻信号辅助判断体动和接触状态变化。

下一步建议的硬件方向为：在保留现有长条式复合传感器结构的基础上，采用两条或多条同款模组在床垫下平行铺设，并通过同一采集模块同步采集多路原始信号。

推荐基础布置：

```text
床头
┌────────────────────────────┐
│ Strip 1：胸背/胸廓区域       │
│                            │
│ Strip 2：腹腰/腹部区域       │
│                            │
│ Strip 3：臀胯区域，可选       │
└────────────────────────────┘
床尾
```

其中 Strip 1 主要捕捉胸式呼吸，Strip 2 捕捉腹式呼吸和位置变化，Strip 3 可选，主要用于体动、离床和姿态辅助。

## 2. 相关技术概况

### 2.1 商业产品布置

商业化床垫式睡眠监测产品多数采用单条、单垫或单板式非接触传感器，通常布置在胸背、心脏或躯干区域。例如：

- TE 的 Sleep Monitor Strip 是约 800 mm 的 PVDF 睡眠监测条带，用于呼吸、心跳和体动检测。
- Withings Sleep Analyzer 采用床垫下方气动/压力式传感结构，用于心率、呼吸、体动和睡眠呼吸暂停筛查。
- Emfit QS、EarlySense 等产品也采用床垫下方非接触传感器，主要放在胸背或躯干区域。

这说明单条/单垫结构在工程上可行，具有低成本、易安装和产品化成熟度高的优势。

### 2.2 论文中的阵列和信号处理

与呼吸事件相关的论文和研究中，多通道或阵列方案也较常见，主要用于提高鲁棒性和通道选择能力。

典型方向包括：

- PVDF 4x1 阵列：Hwang 等使用 PVDF 阵列进行非约束睡眠呼吸暂停监测，通过 PVDF 信号变化检测 apnea 事件，并与 PSG 的 AHI 进行对比。
- 多个压电陶瓷传感器阵列：Peng 等使用 18 个压电陶瓷传感器布置在床垫下，通过选择最敏感通道提取呼吸率和心率。
- 高密压力矩阵：用于体位、压力分布、体动和呼吸频率估计。

常见信号处理方法包括：

- 低通、带通、陷波滤波。
- 小波分解、EMD/EEMD。
- PCA、自适应 PCA、ICA 或盲源分离。
- 方差最大、能量最大或频带能量最大通道选择。
- Hilbert 包络、动态平滑、峰谷检测。
- 体动检测、体动段剔除。
- 机器学习分类。

这些方法大多属于已知技术，单独作为专利创新点的空间有限。

### 2.3 相关专利概况

强相关专利已经覆盖了以下方向：

- 床垫下或床垫内布置压电、PVDF、压力、气囊或压阻传感器。
- 胸背、胸腹或躯干区域布置传感器。
- 多个压力传感板、微动传感器阵列、PVDF 多点阵列。
- 利用方差、能量、波形幅度、频域特征进行呼吸/心率/体动检测。
- 压力阵列用于睡姿、离床、体动和生命体征监测。

因此，若仅写“床垫下放置压电传感器”或“采用阵列检测呼吸”，新颖性和创造性风险较高。更适合从复合传感结构、同步多条布置、压阻门控、动态基线和呼吸事件二次验证流程入手。

## 3. 现有技术痛点

面向呼吸事件监测，单纯使用压电/PVDF 信号存在以下问题：

1. PVDF 幅度受睡姿、体型、床垫厚度和接触位置影响较大。
2. 呼吸幅度下降可能是真实低通气，也可能只是身体压点改变或局部耦合变差。
3. 体动、翻身、上床、离床会产生大幅瞬态信号，容易干扰呼吸事件判断。
4. 固定 30 s 窗口或滑动窗口可以缓解事件被切分的问题，但仍然容易把事件起止时间、体动恢复期和姿态变化后的基线变化混在同一套统计窗口内。
5. 简单选择方差最大或能量最大的通道容易选到体动通道，而不一定是最佳呼吸通道。
6. 单条胸背位传感器对腹式呼吸、侧卧、位置偏移的鲁棒性有限。

因此，呼吸事件监测的关键不是单纯提高传感器数量，而是建立能区分以下情况的信号处理流程：

```text
真实呼吸暂停/低通气
vs
体位变化导致的 PVDF 幅度变化
vs
接触位置变化导致的耦合变化
vs
体动或离床造成的伪事件
```

## 4. 拟提出的核心方案

本项目拟提出一种基于压电-压阻复合条带阵列的非接触式呼吸事件监测系统和方法。

系统包括：

- 至少一条压电-压阻复合长条传感器。
- 优选两条或多条复合长条传感器，沿床头到床尾方向间隔平行布置。
- 同一采集模块同步采集每条传感器的压电信号和压阻信号。该点更适合作为系统实现约束或从属权利要求支撑点，而不是单独作为核心创新。
- 数据处理模块根据压阻接触状态、压电呼吸波形、多条信号一致性和体动分段结果，输出呼吸事件候选和可信度。

基础数据格式建议为：

```text
sample_index,
strip1_pvdf, strip1_pr,
strip2_pvdf, strip2_pr,
...
```

## 5. 可写专利创新点

### 创新点一：压电-压阻复合长条的多条平行布置

现有方案多为单一压电传感器、压力阵列或独立压力传感器。本方案采用压电和压阻集成于同一长条模组，每条模组同时输出动态呼吸信号和静态/准静态接触信号。

多条模组在床垫下按胸背、腹腰、臀胯等纵向区域平行布置，不追求高密压力成像，而是形成低密度条带式阵列。

该结构的优点：

- 保留 800 mm 长条横向覆盖能力。
- 增加胸背和腹腰的纵向呼吸信息。
- 硬件复杂度低于高密阵列。
- 更适合呼吸事件而非单纯压力成像。

可强调的区别点：

```text
不是普通 PVDF 阵列；
不是普通压力矩阵；
而是每条均包含 PVDF 动态信号和压阻接触信号的复合条带阵列。
```

### 支撑点二：同一采集模块同步采集多条复合条带

若两条传感器分别使用独立采集板，会出现启动时间不同、MCU 时钟漂移、串口缓冲延迟和丢包后难以对齐等问题。本方案要求由同一采集模块对多条传感器进行统一时钟采样，并输出统一 sample_index 或时间戳。

该点对呼吸事件监测非常重要，因为暂停或低通气需要判断不同区域信号是否同步变弱。但同步采集本身属于多通道系统的常规要求，不宜单独作为核心创造性来源；它的价值在于支撑后续“胸腹一致性判断、压阻门控、事件二次验证”等算法流程成立。

可写为：

- 统一采样时钟。
- 统一采样序号。
- 多条 PVDF 和压阻信号同步输出。
- 可根据同步数据计算胸腹一致性、相位关系和事件同时性。

### 创新点三：压阻接触状态用于 PVDF 呼吸事件判别门控

传统 PVDF 方案主要依据压电信号幅值、方差或频域能量判断呼吸事件。本方案将压阻信号作为呼吸事件判别的门控条件。

判别逻辑示例：

```text
PVDF 幅度下降 + 压阻稳定 + 多条同步下降
→ 疑似呼吸暂停/低通气

PVDF 幅度下降 + 压阻明显变化
→ 疑似体位/接触变化，不直接判定为低通气

PVDF 瞬态突变 + 压阻突变
→ 标记为体动段，暂停呼吸事件判断

PVDF 单条下降 + 另一条正常
→ 优先认为局部耦合变化或姿态变化
```

这一点是现有算法基础中最适合专利化的部分。

### 创新点四：以体动边界作为基线重置点，并在稳定段内进行事件级检测

常见方法采用固定长度窗口或滑动窗口计算 SQI、呼吸率或事件特征。多个重叠窗口可以减轻事件被单个窗口切开的影响，因此“避免切碎事件”不应作为唯一创新点。本方案更准确的思路是：先利用 PVDF 动态变化和压阻接触变化检测体动边界，将体动边界作为姿态/接触状态可能改变的基线重置点；在两个体动边界之间的稳定段内，再采用滑动窗口、逐呼吸周期或事件级方法检测呼吸事件。

在稳定段内再进行：

- 呼吸波形提取。
- 局部基线建立。
- 呼吸暂停/低通气候选检测。
- 事件二次验证。

若两次体动之间持续几十分钟，不将整段作为一个统计窗口处理，而是把它作为一个“接触状态相对稳定的分析域”。在该分析域内继续使用短窗口、滚动基线、逐呼吸周期峰谷检测或候选事件搜索；当压阻接触状态、通道质量或呼吸幅度分布发生持续漂移时，可在段内触发子段划分或局部基线更新。

优势：

- 将体动边界作为基线重置点，避免体动前后的 PVDF 幅度直接混用。
- 避免将体动恢复期、接触重建期误判为呼吸事件。
- 在长稳定段内保留滑动窗口和逐呼吸周期分析能力，既不依赖单个固定窗口，也不把几十分钟数据整体合并判断。

可写成流程：

```text
原始 PVDF/PR 信号
→ 体动候选检测
→ 压阻接触验证
→ 体动段扩展与合并
→ 稳定分析域划分
→ 域内滚动基线/逐呼吸周期/事件级检测
```

### 创新点五：基于压阻状态的动态呼吸基线更新

低通气判断依赖幅度下降比例，但 PVDF 幅度不是稳定潮气量代理量，会受接触状态影响。

本方案不采用全夜固定基线，而是在满足以下条件的片段中建立局部基线：

- 压阻接触状态稳定。
- 无体动。
- PVDF 呼吸周期稳定。
- 当前条带质量评分高。

当压阻信号显示接触状态改变时，系统重建局部基线，而不直接沿用旧姿态下的 PVDF 幅度基线。

该点可用于解决“翻身后 PVDF 幅度变小被误判为低通气”的问题。

### 创新点六：融合压电呼吸质量和压阻稳定性的通道质量评分

已有论文常用方差最大或能量最大选择通道，但体动时方差和能量也会最大。本方案提出结合多类指标的通道质量评分。

质量评分可包括：

```text
呼吸频段能量
呼吸周期稳定性
峰谷间隔稳定性
呼吸包络稳定性
压阻接触稳定性
ADC 坏点/饱和比例
体动突变分数
多条信号一致性
```

质量评分用于：

- 选择主呼吸通道。
- 判断是否需要切换通道。
- 判断事件候选是否可信。
- 生成每个呼吸事件的置信度。

区别于简单的最大方差或最大能量通道选择。

### 创新点七：胸背-腹腰双条一致性用于呼吸事件二次验证

对于两条平行布置的传感器：

- 胸背条反映胸廓运动。
- 腹腰条反映腹式呼吸和腹部运动。

事件二次验证可以包括：

```text
两条 PVDF 呼吸幅度同时下降，且压阻稳定
→ 事件可信度升高

一条下降，另一条保持正常呼吸
→ 事件可信度降低，优先解释为局部接触变化

胸背和腹腰信号相位/幅度关系异常
→ 输出疑似阻塞性呼吸事件或胸腹不同步提示

两条压阻均发生突变
→ 判为体动或姿态变化，事件检测暂停
```

该点比单条 PVDF 更适合呼吸暂停/低通气监测。

### 创新点八：呼吸事件候选与验证分离的两级判定流程

本方案将呼吸事件检测分为两级：

第一级：由 PVDF 呼吸波形产生事件候选。

- 呼吸峰谷消失。
- 呼吸包络持续下降。
- 呼吸周期异常延长。
- 呼吸频段能量下降。

第二级：由压阻和多条一致性进行验证。

- 压阻是否稳定。
- 是否存在体动。
- 是否为多条同步下降。
- 当前通道质量是否足够。
- 是否需要重建基线。

最终输出：

```text
事件类型：疑似暂停 / 疑似低通气 / 体动伪事件 / 低可信片段
起止时间
持续时间
主通道
辅助通道一致性
压阻稳定性
置信度
```

## 6. 建议权利要求方向

### 独立权利要求 1：系统权利要求

一种非接触式呼吸事件监测系统，包括：

- 至少两条压电-压阻复合长条传感器；
- 所述复合长条传感器沿床垫宽度方向延伸，并沿床垫长度方向间隔布置；
- 同一采集模块，用于同步采集每条复合长条传感器的压电信号和压阻信号；
- 处理模块，用于根据压电信号生成呼吸事件候选，并根据压阻信号和多条传感器之间的一致性对所述呼吸事件候选进行验证。

### 独立权利要求 2：方法权利要求

一种呼吸事件监测方法，包括：

1. 同步获取多条复合长条传感器的压电信号和压阻信号；
2. 根据压电信号和压阻信号检测体动段，并将体动边界作为接触状态或姿态状态可能改变的分界点；
3. 在非体动稳定分析域内，通过滑动窗口、逐呼吸周期或滚动统计建立局部呼吸基线；
4. 根据压电呼吸波形生成呼吸事件候选；
5. 根据压阻接触状态、多条信号同步变化关系和通道质量评分对呼吸事件候选进行二次验证；
6. 输出呼吸事件类型、起止时间和置信度。

### 从属权利要求可覆盖的点

- 传感器为两条，分别布置在胸背区域和腹腰区域。
- 传感器为三条，第三条布置在臀胯区域，用于体动或离床辅助。
- 每条传感器包括 PVDF 压电薄膜和压阻薄膜。
- 采集模块输出统一 sample_index。
- 通道质量评分包含呼吸频段能量、周期稳定性、压阻稳定性和体动分数。
- 当压阻接触状态改变时，重新建立局部呼吸基线。
- 当仅单条压电信号下降而另一条保持稳定呼吸时，降低呼吸事件置信度。
- 当多条压电信号同时下降且压阻稳定时，提高呼吸事件置信度。
- 体动段内暂停呼吸暂停/低通气判断。
- 呼吸事件候选包括呼吸峰谷消失、呼吸包络下降、呼吸周期延长或呼吸频带能量下降。

## 7. 与已有技术的区别

本方案与已有技术的区别可以概括为：

| 已有技术 | 本方案区别 |
|---|---|
| 单条 PVDF 或单垫传感器 | 多条压电-压阻复合长条，覆盖胸背和腹腰 |
| 普通压力阵列 | 不追求高密压力成像，而是低密条带式呼吸事件监测 |
| 仅用 PVDF 幅值/方差判断事件 | 引入压阻接触状态作为事件门控和二次验证 |
| 方差最大/能量最大通道选择 | 引入呼吸周期稳定性、压阻稳定性、体动分数的质量评分 |
| 固定窗口或滑动窗口检测 | 以体动边界作为基线重置点，在稳定分析域内结合滚动基线、逐呼吸周期和事件级检测 |
| 多板独立采集 | 同一采集模块同步输出多条原始信号，作为胸腹一致性和事件二次验证的系统支撑条件 |
| 只输出呼吸率 | 面向暂停/低通气等呼吸事件，输出事件类型和置信度 |

## 8. 实验验证建议

为了支撑专利和后续论文，建议采集以下数据：

### 8.1 单条传感器基础精度

- 呼吸率与参考设备的误差。
- 有效呼吸波形覆盖率。
- 体动段识别准确性。
- 不同睡姿下 PVDF 幅度和压阻基线变化。

### 8.2 两条平行布置收益

- 胸背条和腹腰条呼吸质量对比。
- 单条低质量时另一条是否可用。
- 两条同步下降与真实呼吸事件的相关性。
- 体位变化时压阻信号能否解释 PVDF 幅度变化。

### 8.3 呼吸事件验证

- 人工模拟屏气/浅呼吸实验。
- 正常呼吸、翻身、离床、侧卧、偏位等干扰实验。
- 与参考呼吸带、鼻气流或 PSG 数据对齐。
- 对比单条算法与两条同步算法的误报率和漏报率。

可选题目：

## 11. 初步结论

滤波、小波、EMD、PCA、动态平滑、包络提取和机器学习分类等基础算法大多已有公开使用，单独作为创新点较弱。

本项目更适合围绕以下组合形成专利创新：

```text
压电-压阻复合长条传感器
+ 多条胸背/腹腰平行布置
+ 同一模块同步采集作为系统支撑条件
+ 压阻接触状态门控
+ 体动分段后检测呼吸事件
+ 动态呼吸基线
+ 多条胸腹一致性二次验证
+ 通道质量评分和事件置信度输出
```

该方案既贴合现有实物和算法基础，也能与普通单条传感器、高密压力阵列、单纯 PVDF 阵列和常规信号处理方法形成区别。
